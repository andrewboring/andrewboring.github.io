This whole site is about me. Why should I summarize it on one page? Let's talk about the site, instead.

This site is generated using the Pelican static site generator using Markdown source files and Jinja2 templates. The content is served from Github Pages.

Site theme was originally adapted from monospace, but later restyled using Bootstrap.

The content is organized to showcase my work, by aggregating content from other sources I contribute to. I also post occasional weblog content directly here.
